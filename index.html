Introduction to Human Computer Interaction

Human Computer Interaction (HCI) is the study of how people interact with computer systems. It focuses on designing interfaces that are usable, efficient, and accessible for users.

Interaction Models

Interaction models are used in HCI to understand and evaluate how users perform tasks using a system. These models help designers improve usability and reduce user errors.

Norman, GOMS, and KLM Overview

Norman’s Action Cycle explains how users form goals and execute actions. GOMS (Goals, Operators, Methods, and Selection rules) analyzes user tasks, while KLM (Keystroke-Level Model) predicts task completion time using low-level actions.
